How To

Here you will find information on how to apply the patches to your Ultima Online Client UOP Files (post 7.0.24)

Getting Started & Backups!

1: Download Mulpatcher from here:

http://varan.uodev.de/

Download UOFiddler from here:
http://uofiddler.polserver.com/


Download UOFiddler UOP Extractor Plugin from here:
http://www.runuo.com/community/resources/uofiddler-uop-extractor-plugin.125/
copy the plugin into the plugin folder of UOFiddler, enable the plugin in UOFiddler itself
(see images at bottom if you have problems)

Incase the Downloads went down, you can find the files here:
UOFiddler4.6_withUOP.zip

You can also use LegacyMULConverter from cbnolok to convert UOP<->MUL:
DirectLink



2: Make a Backup of the following files

X:\Ultima Online Classic\artLegacyMUL.uop
X:\Ultima Online Classic\gumpartLegacyMUL.uop

I suggest you make a new folder called "Backup"
and make a copy of the files there

Extract MUL file from UOP Files

Open UOFiddler, make sure to load the plugin, go to the UOP Packer Tab
In the "One File" subtab:
(you need to convert both files, after another)

Output MUL: art.mul / gumpart.mul
Output IDX: artidx.mul / gumpidx.mul
Input type: ArtLegacyMUL / GumpartLegacyMUL
Input UOP: artLegacyMUL.uop / gumpartLegacyMUL.uop

LegacyMULCL-N: Use a CMD into the folder. This example used C:\TEMP where the .uop files are, adjust to your enviroment. Type:
LegacyMULCL-N.exe -x "C:\TEMP"



Now should have converted your UOP to a patchable MUL file!

Start Mulpatcher and loading the Art/Gump files

3: Start Mulpatcher

4: In the [Settings]-Tab choose the location for
artidx.mul and art.mul -> Click the "Load" button
(may take some time to load, program wil not
respond while loading)

5: In the [Settings]-Tab choose the location for
gumpart.mul and gumpidx.mul -> Click the "Load"
button (may take some time to load, program wil
not respond while loading)

Using the different patches

6: Change to the [Features]-Tab

------------------------------------------------
REPEAT Step 7-9 FOR EACH PATCH YOU WANT TO USE
------------------------------------------------

7: Load the *.txt file from the patch-directory
(extracted .zip you download from "finished")
you want to use into "Autopatch" at top left:

choose Art(S) for the following patches:
cleanfloor, customfields, easydungeontraps,
notrees, nosailboat, nostaticroofs, staticfields
and easyeb, IPY brambles fix, IPY lotteryfix,
BetterRegs, BigKey, EasyDungeonTrapsFix,
GlowSwitch, SmallGold, SmallRobeNCloak



choose Gumps for the following patches:
journal, statusbars, CustomLogon, GreenBags,
TradeWindow

8: hit the start button, this will apply the
patch you chose.

9: A progress-bar should come up, stating how
many pictures were replaced. Confirm with OK.

------------------------------------------------
REPEAT Step 7-9 FOR EACH PATCH YOU WANT TO USE
------------------------------------------------

Saving the patched MUL files

10: Change back to [Settings]-Tab
11: Click the Save button and save your patched
artidx.mul/art.mul or gumpart.mul/gumpidx.mul


Repack UOP Files from patched MUL Files

Open UOFiddler, make sure to load the plugin, go to the UOP Packer Tab
In the "One File" subtab:
(you need to convert both files, after another)

Input MUL: art.mul / gumpart.mul
Input IDX: artidx.mul / gumpidx.mul
Input type: ArtLegacyMUL / GumpartLegacyMUL
Output UOP: artLegacyMUL.uop / gumpartLegacyMUL.uop

LegacyMULCL-N: Use a CMD into the folder. This example used C:\TEMP where the patched .mul files are, adjust to your enviroment. NOTE: delete any .uop files in C:\TEMP or repack fails because file exists! Type:
LegacyMULCL-N.exe "C:\TEMP"

Now should have repacked UOP files from the patched MUL Files

Replace the UOP files in your UO Classic Client folder!

Start Ultima Online Classic

Go check it out!

NOTES / KNOWN ISSUES

Note: Dont use customfields and staticfields together, one will overwrite the other

Note2: Remove the cleanfloor patch if you want to place a house. I am not sure if it has an inpact.

Note3: You need to unpack your .uop files into .mul files, apply the mods with mulpatcher, and then repack the modded .mul files into .uop files, copy and replace them in your UO folder

Note4: If there is no UOP tab in UOFiddler, make sure its checked under Plugins AND restart UOFiddler
